자바스크립트 : 정적인 웹문서에 동작을 부여하는 언어

API :
- Application Programming Interface
- HTML에서 제공하는 기능(함수)의 모임
- Geolocation / Canvas / Drag&Drop
- 자스 라이브러리 : JQuery / Angular.JS / React.JS / Node.JS / PhoneGap 등 

개발자 도구[F12] 패널
Elements : HTML 요소에 적용된 스타일을 검사
Consloe : 자바스크립트 오류 체크 / 디버깅Debugging
Source : 브라우저가 자스 소스를 파싱Parsing 해오는 과정을 보여줌. 소스 패널도 오류체크와 디버깅 가능

* 파싱Parsing : 저장된 값을 원하는 형식의 값으로 가공해 읽어오는 것
* 디버깅Debugging : 프로그래머가 실수로 잘못 코딩한 프로그램을 실행하면 오류가 발생 한다. 이런 현상을 Bug라 부르고, 이런 Bug를 검사하고 수정하는 작업을 디버깅이라고 함.

웹 에디터 : 에디트플러스/ 노트패드++/ 아톰/ 서브라임 텍스트/ 브라켓/ 웹스톰/ 비쥬얼 스튜디오 코드 등

비쥬얼 스튜디오 환경 구성
뷰 바 : 사이드 바의 기능(탐색/검색/깃/디버그/확장) 변경 가능
사이드바 : 프로젝트 작업을 수행하는 동안 탐색, 검색, 깃, 디버그, 확장 기능으로 바꾸며 사용
편집기 그룹 : 소스를 편집하는 영역. 최대 3개까지 창 분할 가능.
패널 : 출력, 디버그 정보, 오류 및 경고 등 표시
상태바 : 현재 프로젝트와 편집하고 있는 파일의 정보를 제공
